# Skywatcher
A weather forecasting app using AJAX and API.

## Technologies Used:
- HTML5
- CSS3
- JavaScript
  - Typed.js (www.mattboldt.com)
- jQuery
- Twitter Bootstrap
- AJAX
- API (Open Weather Map)
